# Turnitin-CLI

A commandline interface to https://www.turnitin.com

# Setup

``pip install turnitin-cli``

# Usage

``python -m turnitin_cli``

Default keybindings:

| Key   | Function  |
| ----- | --------- |
| ``,`` | Menu down |
| ``.`` | Menu up   |
| ``q`` | Quit      |

These keybindings can be changed in the configuration file located at ``$HOME/tiiconfig.ini``.
